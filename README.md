# Mini Processor for BCD Arithmetic Operations

This project was developed as part of the Digital Electronics module (Year 1 – Semester 2) at SLIIT.

## 📌 Overview
The project focuses on designing and simulating a mini processor capable of performing arithmetic operations on Binary Coded Decimal (BCD) inputs.

The processor accepts two 1-digit BCD numbers and performs operations based on control signals.

## ⚙️ Supported Operations
- Addition (00)
- Subtraction (01)
- Multiplication (10)

## 🔢 Output
- 2-digit BCD result (00–99)
- Overflow indicator for values exceeding 99

## 🧩 Components Used
- BCD Adder with correction logic
- BCD Subtractor using two’s complement
- BCD Multiplier (shift-add/array method)
- Multiplexer (for operation selection)
- Registers for input/output storage

## 🛠️ Tools & Technologies
- Proteus Simulator

## 🧪 Testing
The system was tested using multiple input combinations to verify:
- Correct arithmetic operations
- Accurate BCD output
- Proper overflow detection
